# 💍 Digital Wedding Invitation Website

A beautifully crafted, responsive, and culturally accurate digital wedding invitation built for the auspicious marriage ceremony of **Huokaing Thara & Khen Lyda**. This interface balances traditional Cambodian royal aesthetics with high-performance modern web standards.

---

## ✨ Features

*   **Premium Khmer Typography:** Leverages Google Web Fonts explicitly optimized for rendering `Moul` and `Hanuman` font-families cleanly across all viewports.
*   **Cultural Accuracy:** Features structural honors, traditional parent placeholder listings, and a formal lunar calendar date structure native to high-end Cambodian invitations.
*   **Rich Graphic Integration:** Built-in structural blocks for custom high-resolution traditional golden ornaments and background framing headers/footers.
*   **Interactive Navigation:** Fully responsive action button that dynamically interfaces with the custom mapping deployment page (`https://tharahuokaing.github.io/restaurant1/`).
*   **Mobile-First Responsive Design:** Optimized layouts with fluid CSS Grid system breakpoints designed to look impeccable on iOS and Android devices.
